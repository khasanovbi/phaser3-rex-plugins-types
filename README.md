Types for [phaser3-rex-plugins](https://github.com/rexrainbow/phaser3-rex-notes)

## Getting Started

Install the module:

```
npm i --save phaser3-rex-plugins-types
```

Add to your `tsconfig.json` next lines:
```
    "include": [
        "**/*",
        "node_modules/phaser3-rex-plugins-types"
    ]
```
